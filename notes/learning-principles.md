# Learning Principles

## Core Rules

1. Write code every day.
2. Do not only read or watch.
3. Build small projects after learning concepts.
4. Keep notes simple and practical.
5. Commit progress regularly.
6. Refactor old code as understanding improves.
7. Focus on clarity before cleverness.

## Daily Flow

1. Learn the topic.
2. Write notes.
3. Solve beginner exercises.
4. Solve intermediate exercises.
5. Try advanced exercises.
6. Build a small project.
7. Update progress.
8. Commit to GitHub.
