# Language Playground

This repository is my personal programming language learning playground.

The goal is to learn programming languages through structured notes, hands-on exercises, mini-projects, and consistent progress tracking.

This repo is not only for syntax learning. The main goal is to build practical programming skills through daily practice, problem solving, documentation, and project-based learning.

## Main Rule

Learn by writing code daily, solving problems, building small projects, and documenting what I understand.

## Daily Learning Flow

1. Read the topic overview.
2. Study the notes.
3. Write code examples.
4. Solve beginner exercises.
5. Solve intermediate exercises.
6. Try advanced exercises.
7. Build small projects.
8. Update progress.
9. Commit to GitHub.

## Long-Term Goal

This repository should become a clear proof of my learning consistency, coding growth, documentation habit, and practical programming mindset.
